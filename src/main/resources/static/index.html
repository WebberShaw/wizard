关于软件

tDDNS-Master采用跨品台语言Java编写，可以同时支持PC、Mac、Linux。软件调用了腾讯云官方的接口实现动态域名解析，只需在腾讯云注册域名，或将域名解析转移到腾讯云名下，并在腾讯云官网获取SecretId和SecretKey即可将您的域名实时解析到您的公网IP，并可以通过Server 酱提供的服务自动将解析IP变更成功或失败的信息推送到您的手机中。软件可以支持超时重试功能，重试时长和间隔均可自定义。
